1. Introducción

Bienvenido a BarberFlow+, el software de gestión integral diseñado para modernizar y optimizar la administración de barberías y peluquerías.

Este manual interactivo te guiará a través de todas las funcionalidades de la aplicación, desde la configuración inicial y la gestión diaria hasta la generación de reportes avanzados.

2. Primeros Pasos

2.1. Selección de Tema Visual

Al ingresar por primera vez, podrás elegir tu experiencia visual preferida entre Modo Claro y Modo Oscuro.

2.2. Creación de una Cuenta y Acceso

Si eres un cliente nuevo, haz clic en "Regístrate aquí". Si ya tienes un perfil, introduce tu correo y contraseña para acceder.

3. Módulos Principales

3.1. Dashboard (Panel Principal)

Es tu centro de control. Ofrece un resumen vital del estado de tu negocio en tiempo real.

Acciones Rápidas (Proceso Detallado)

Acceso directo a las operaciones más comunes. Haz clic en cada acción para expandir.

  1. Haz clic para abrir la ventana "Registrar Gasto".
  2. Descripción: Escribe un detalle claro del gasto.
  3. Monto: Ingresa el valor total.
  4. Tipo de Gasto: Selecciona una categoría (Gasto Operativo, Costo Fijo, etc.).
  5. Haz clic en "Guardar Gasto" para registrarlo en tus finanzas.
  1. Usa esta opción para añadir un nuevo producto a tu inventario.
  2. Completa los campos: Nombre, Precio de Venta, Costo de Compra y **Stock Mínimo**.
  3. Haz clic en "Guardar Producto".
  1. Utiliza esta acción cuando recibas mercancía.
  2. Producto: Selecciona el producto del menú.
  3. Cantidad Comprada: Ingresa el número de unidades recibidas.
  4. Costo Total: Ingresa el valor total que pagaste por esa compra.
  5. Haz clic en "Registrar Compra" para actualizar el stock.

  2. Producto: Selecciona el producto vendido.
  3. Cantidad: Ingresa el número de unidades.
  4. Haz clic en "Confirmar" para registrar la venta y descontar el stock.

3.2. Citas (Proceso Detallado)

El corazón de tu negocio. Este módulo presenta un calendario interactivo para una gestión completa del agendamiento.

  1. Navegación: Al entrar al módulo, verás un calendario semanal. Usa los botones para moverte a semanas anteriores o posteriores.
  2. Filtrar por Barbero: En la parte superior, usa el menú desplegable "Ver Calendario de:" para ver la agenda de un barbero específico.
  1. Seleccionar Horario: Haz clic en un espacio de tiempo vacío del calendario. Esto abrirá el formulario "Nueva Cita".
  2. Completar Formulario: Rellena los campos Cliente, Barbero, Servicio, Fecha/Hora y Duración.
  3. Guardar: Haz clic en "Guardar Cita". La nueva cita aparecerá en el calendario.

Haz clic sobre cualquier cita ya agendada. Se abrirá la ventana "Detalle de la Cita" donde podrás:

  • Registrar Pago: Abre la ventana de pago para confirmar la transacción. El color de la cita cambiará.
  • Cancelar Cita: Anula la cita. Podrás añadir un motivo. La cita se volverá gris.

4. Funciones Especiales

4.1. Asistente OpenAI (Proceso Detallado)

El ícono 🤖 te da acceso a un asistente de IA para "conversar" con tus datos. Haz preguntas en lenguaje normal y obtén respuestas precisas sin navegar por menús.

El Proceso Técnico (Paso a Paso)

  1. Tu Pregunta: Escribes tu consulta en lenguaje natural (ej: "ingresos de ayer").
  2. Traducción Inteligente: Tu pregunta se envía a un modelo avanzado de IA que actúa como un programador experto. Este "programador" conoce la estructura de tu base de datos y traduce tu pregunta a un código de consulta técnico y preciso (SQL).
  3. Verificación de Seguridad: Antes de ejecutar nada, BarberFlow+ revisa el código SQL generado por la IA. Se asegura de que sea una consulta de **solo lectura** (`SELECT`). Cualquier intento de modificar o borrar datos (`UPDATE`, `DELETE`, etc.) es bloqueado automáticamente. **Tus datos están siempre seguros.**
  4. Ejecución y Resultados: Una vez verificada, la consulta segura se ejecuta en tu base de datos y los resultados se muestran en una tabla clara.

Ejemplos de Consultas que Puedes Realizar

// Sobre Citas y Agenda:

"Muéstrame las citas para mañana"

"¿Cuántas citas atendió el barbero Juan Pérez la semana pasada?"


// Sobre Clientes:

"Encuentra el número de teléfono del cliente Carlos Rivas"

"¿Qué clientes cumplen años esta semana?"


// Sobre Finanzas:

"¿Cuál fue el total de ingresos del día de ayer?"

"Lista de los pagos realizados hoy con Nequi"
